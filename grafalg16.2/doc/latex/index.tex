Estas são instruções gerais para trabalhos práticos de programação. Notação\+:


\begin{DoxyEnumerate}
\item {\ttfamily }[Y]\+: número do trabalho. Deve ser substituído por 1 para o trabalho 1, 2 para o trabalho 2, e assim por diante.
\item {\ttfamily }[L\+A\+B\+G\+A\+D\+IR]\+: diretório para abrigar os arquivos dos trabalhos. Deve ser substituído pelo nome de escolha pessoal para abrigar os arquivos relativos aos trabalhos de Laboratório de Grafos e Algoritmos.

Os diretórios {\ttfamily grafalg16.\+2/lib} e {\ttfamily grafalg16.\+2/bin} destinam-\/se a abrigar códigos gerados como produtos da compilação de {\ttfamily trab}[Y]\+\_\+0000000001.\+c. O diretório {\ttfamily doc} contém esta documentação. Os arquivos gerados na compilação são o código-\/objeto {\ttfamily grafalg16.\+2/lib/trab}[Y]\+\_\+0000000001\+\_\+[A\+R\+CH].o e o executável {\ttfamily grafalg16.\+2/bin/autor\+\_\+}[X\+X\+X\+X\+X\+X\+X\+X\+XX]\+\_\+[A\+R\+CH].\section{Preparação}\label{index_preparacao}
Os passos para a preparação de um trabalho são os seguintes\+:


\begin{DoxyEnumerate}
\item Recuperar o arquivo compactado do trabalho, denominado {\ttfamily trab}[Y].tgz.
\item Crie um diretório para abrigar os arquivos. Nesta documentação, refere-\/se a esse diretório pelo símbolo [L\+A\+B\+G\+A\+D\+IR].
\item Descompactar o arquivo {\ttfamily trab}[Y]-\/tar.\+gz em {\ttfamily }[L\+A\+B\+G\+A\+D\+IR], o que criará uma estrutura de diretórios como a indicada acima.
\item Renomear o arquivo {\ttfamily trab}[Y]\+\_\+0000000001.\+c, substituindo o {\ttfamily 0000000001} pelo seu número de matrícula {\ttfamily }[X\+X\+X\+X\+X\+X\+X\+X\+XX] (somente os números, omitindo o \char`\"{}-\/\char`\"{}).
\item Renomear os diretórios {\ttfamily grafalg16.\+2/trabs/autor\+\_\+0000000001} e {\ttfamily grafalg16.\+2/lib/autor\+\_\+0000000001}, substituindo o {\ttfamily 0000000001} pelo seu número de matrícula (somente os números, omitindo o \char`\"{}-\/\char`\"{}).
\item Copiar os seus arquivos {\ttfamily trab1\+\_\+}[X\+X\+X\+X\+X\+X\+X\+X\+XX].c, ..., {\ttfamily trab}[Y-\/1]\+\_\+[X\+X\+X\+X\+X\+X\+X\+X\+XX].c para o diretório {\ttfamily grafalg16.\+2/trabs/autor\+\_\+}[X\+X\+X\+X\+X\+X\+X\+X\+XX].
\item Editar o arquivo {\ttfamily trab}[Y]\+\_\+0000000001.\+c usando um editor de textos a fim de\+:
\begin{DoxyEnumerate}
\item Na função {\ttfamily ident}[Y](), substituir {\ttfamily 0000000001} pelo seu número de matrícula (somente os números, omitindo o \char`\"{}-\/\char`\"{}).
\item Implementar o código de cada uma das demais funções.
\end{DoxyEnumerate}
\item Compilar.
\item Executar.
\end{DoxyEnumerate}\subsection{Programação}\label{index_Programação}
As funções a serem implementadas estão listadas no arquivo {\ttfamily trab}[Y]\+\_\+0000000001.\+c. Os detalhes das suas especificações estão disponíveis no link {\ttfamily trab}[Y].h na aba {\tt Arquivos}.

Algumas das funções em {\ttfamily trab}[Y]\+\_\+0000000001.\+c podem já estar implementadas. Esses casos devem ser tomados como exemplos de utilização de funções de {\ttfamily graph.\+h} e {\ttfamily \doxyref{direct/bitmap.\+h}{pag.}{bitmap_8h}}.\subsection{Compilação}\label{index_Compilação}

\begin{DoxyEnumerate}
\item Editar o arquivo {\ttfamily grafalg16.\+2/trabs/\+Makefile}, alterando as definições das seguintes variáveis\+: \begin{DoxyVerb} WKSPCDIR:=$(HOME) -->  WKSPCDIR:=$(HOME)/[Caminho até o diretório [LABGADIR]]
 SRCDIR:=$(WKSPCDIR) --> SRCDIR:=$(WKSPCDIR)/[LABGADIR]
\end{DoxyVerb}

\item Executar o comando {\ttfamily make} {\ttfamily alltrabs} no diretório {\ttfamily }[L\+A\+B\+G\+A\+D\+IR]/grafalg16.2/trabs.
\end{DoxyEnumerate}

Caso seja necessário, o comando {\ttfamily make} {\ttfamily clearall} quando executado no diretório {\ttfamily }[L\+A\+B\+G\+A\+D\+IR]/grafalg16.2/trabs remove todos os arquivos gerados em uma compilação.\subsection{Execução e Testes}\label{index_execucao}
