Descrição Detalhada

Bitmap implementation of sets of integers of the type \(\{ 0, 1, \ldots, n-1 \}\), for a specified integer \(n \geq 0\).

A bitmap is an array of bits to represent sets in which a bit of index i represents element i of the set. For a bitmap B, B[i] = 1 means that element i is in the set; otherwise, i is not.

Bitmaps are defined as arrays of five different types depending on the number of bits per node, namely: unsigned char (8 bits per node), unsigned long int (32 bits per node), and unsigned long long int (64 bits per node). The type currently in use is selected by means of the definition of a macro name, respectively with -D_ARCH8_, -D_ARCH32_, -D_ARCH64_, -D_ARCH128_, or -D_ARCH256_ in the gcc command line.
